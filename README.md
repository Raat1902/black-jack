# Blackjack (Java)

A simple terminal-based **Blackjack** game written in **Java** where you play against the dealer.

## Features
- Full 52-card deck
- Hit / Stand
- Dealer hits until 17+
- Ace logic (11 or 1)
- Win / Loss / Tie detection
